# classification
- Was the overall risk level (low / medium / high) appropriate for the provided data?
- Was the recommendation (approve / review / decline) consistent with the risk level?

# use_of_data
- Did the explanation reference the most important factors from the input data?
- For riskier cases, did it highlight the drivers of risk (for example, recent late payments or very thin margins) instead of giving a generic answer?

# explanation
- Was the explanation clear, concise, and understandable to a non-expert?
- Did it avoid internal jargon and schema references?
